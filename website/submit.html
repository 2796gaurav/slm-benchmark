Submit Your Model

To submit a Small Language Model (SLM) for evaluation, follow these steps:

  1. Fork our GitHub repository.
  2. Create a new YAML file in models/submissions/ with your model details.
  3. Open a Pull Request.
  4. Our automated benchmark runner will test your model and post results in the PR.

Example Submission YAML

model:
  name: "MySLM-1B"
  family: "Llama"
  hf_repo: "username/myslm-1b"
  parameters: "1.1B"
  license: "Apache-2.0"
  quantizations:
    - name: "FP16"
    - name: "Q4_K_M"
  categories:
    - "reasoning"
    - "language"
