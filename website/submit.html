2

Clone & Create a Pull Request

Fork the repository, clone it to your local machine, and add your YAML file to models/submissions/. Then push your changes and create a pull request. Our maintainers will review, validate the model, and merge your submission.

git clone https://github.com/YOUR_USERNAME/slm-benchmark.git
cd slm-benchmark
# Add your YAML to models/submissions/your-model.yaml
git add models/submissions/your-model.yaml
git commit -m "Submit: Your Model Name"
git push origin main
# Then open a PR on GitHub
3

Wait for Evaluation

Our community runners will pick up your submission, validate the model on Hugging Face, run the comprehensive benchmark suite, and update the public leaderboard. Updates typically appear within 24 hours once processing is complete.
